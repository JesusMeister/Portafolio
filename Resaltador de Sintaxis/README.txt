Instrucciones de uso: 
Las dos funciones principales son:
- Secuencial: funcion(x y)
- Paralela: funcionP(x y)

Para uso general, escribir los archivos a resaltar en la lista "leer'()" y ejecutar el programa
Para uso personalizado, realizar una lista con nombres de archivos a resaltar, y una lista con los nombres de los arhivos html a escribir

Los archivos a leer deberan estar en la carpeta de lectura
Los archivos nuevos se escribir en la carpeta html
